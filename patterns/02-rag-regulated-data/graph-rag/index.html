Pattern 02 - GraphRAG

GraphRAG for Regulated Data

Knowledge graph-based retrieval for complex entity relationships. Combines vector similarity with graph traversal for multi-hop reasoning over structured relationships.

When to use this variant
Use Complex entity relationships across documents
Use Multi-hop reasoning ("who reports to whom")
Use Entity disambiguation and linking
Use Questions about connections, hierarchies, networks
Skip Simple factual Q&A (see standard-rag)
Skip Self-correcting retrieval (see agentic-rag)
RAG quality metrics
Retrieval: precision, recall, MRR, NDCG
Generation: groundedness, faithfulness, relevance
Safety: PHI leakage, hallucination rate
Citations: accuracy, completeness, link validity
Ops: latency p50/p99, cache hit rate, cost/query
