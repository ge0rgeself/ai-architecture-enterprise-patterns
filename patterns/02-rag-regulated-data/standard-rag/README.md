# Variant: Standard RAG

**Vector search + chunking + generation with guardrails. The baseline RAG pipeline.**

## Live Demo

**[Launch interactive visualization](https://aman210122.github.io/ai-architecture-enterprise-patterns/patterns/02-rag-regulated-data/standard-rag/index.html)**

Scenarios: Factual query, multi-doc synthesis, sensitive data (PHI), no-results handling, citation-grounded response.

## When to Use

**Use:** Standard document Q&A. Single data source. Citation-grounded responses. Compliance audit trail.

**Skip:** Complex entity relationships (see graph-rag). Self-correcting retrieval (see agentic-rag).
