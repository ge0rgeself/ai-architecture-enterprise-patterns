# Variant: Anonymization Routing

**Route by de-identifiability. Anonymizable data goes to cheap providers.**

## Live Demo
**[Launch](https://aman210122.github.io/ai-architecture-enterprise-patterns/patterns/08-compliance-data-routing/anonymization-routing/index.html)**
