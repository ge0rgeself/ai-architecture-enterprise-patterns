When to use this variant
Use Agents need to call external tools (DB, API, file) (extract, analyze, synthesize)
Use Every tool call must be authorized and audited
Use Tools have side effects that need governance you need to prevent
Use Audit trail of every tool invocation is required
Skip Simple function calling is enough (see function-calling)
Skip No external tools needed (see linear-chain)
