# Variant: Agentic RAG

**Self-correcting retrieval with CRAG/Self-RAG patterns. Re-retrieves on low confidence.**

## Live Demo

**[Launch interactive visualization](https://aman210122.github.io/ai-architecture-enterprise-patterns/patterns/02-rag-regulated-data/agentic-rag/index.html)**

Scenarios: Adaptive retrieval, low-confidence recovery, multi-hop planning, sensitive re-retrieval, cited response.

## When to Use

**Use:** First retrieval may miss context. Multi-hop questions. Complex analysis. Variable retrieval confidence.

**Skip:** Simple factual Q&A (see standard-rag). Entity relationships (see graph-rag).
