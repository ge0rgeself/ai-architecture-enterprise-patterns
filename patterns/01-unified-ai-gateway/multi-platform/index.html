When to use this variant
Use Your org runs Databricks, Azure AI, and Snowflake together
Use Multiple teams consume LLMs independently, need centralized governance
Use Sensitive data (PHI, PCI, PII) requires provider-level routing restrictions
Use Both classic ML and Gen AI workloads coexist on one platform
Skip Single cloud provider only (see variant 01b or 01c)
Skip Budget or team cannot support multi-platform operations
