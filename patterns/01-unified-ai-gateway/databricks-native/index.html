When to use this variant
Use Your org is standardized on Databricks
Use You prefer open-weight models for cost and control
Use Data sovereignty requires all processing in your own environment
Use You want to fine-tune models on proprietary data
Skip You need GPT-4o or Claude level reasoning (see 01a or 01c)
Skip Multiple data platforms required (see 01a Multi-Platform)
Databricks-native advantage
Zero external API dependencies
Full data control (nothing leaves your VPC)
Unity Catalog governs data + models + features + agents
Fine-tune on proprietary data without vendor exposure
Swap models instantly (DBRX to Llama to Mixtral)
~$800/month infrastructure (lower than multi-platform)
