When to use this variant
Use Your org is a Microsoft shop with Azure AD / Entra ID
Use You need M365 Copilot extensibility
Use GPT-4o/4.1 level reasoning is required
Use BAA is available (Azure OpenAI supports HIPAA)
Skip You need open-weight model control (see 01b)
Skip Multi-cloud is a strategic requirement (see 01d)
Azure-native advantage
Managed Identity auth (no API keys to manage)
APIM has native GenAI gateway policies (no custom code)
Content Safety + Prompt Shields built-in
M365 Copilot extensibility for enterprise adoption
Purview DLP for sensitive data governance
~$900/month infrastructure
