# Variant: GraphRAG

**Knowledge graph-based retrieval. Entity resolution + community detection for multi-hop reasoning.**

## Live Demo

**[Launch interactive visualization](https://aman210122.github.io/ai-architecture-enterprise-patterns/patterns/02-rag-regulated-data/graph-rag/index.html)**

Scenarios: Entity queries, relationship traversal, community analysis, sensitive entities, citation-grounded.

## When to Use

**Use:** Complex entity relationships. Multi-hop reasoning. Entity disambiguation. Hierarchies and networks.

**Skip:** Simple factual Q&A (see standard-rag). Self-correcting retrieval (see agentic-rag).
