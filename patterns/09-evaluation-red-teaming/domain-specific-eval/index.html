When to use this variant
Use Need systematic AI quality and safety evaluation (extract, analyze, synthesize)
Use Each agent output must be validated before the next agent sees it
Use Contamination between agents is a risk you need to prevent
Use Regulatory audit requires inter-agent validation evidence
Skip Manual testing is sufficient (early stage)
Skip Need consensus across agents (see consensus)
