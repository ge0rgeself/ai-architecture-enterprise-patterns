Use Your org is a Microsoft shop with Azure AD / Entra ID
Use You need Google Search grounding for factual accuracy
Use Gemini 2.0 Flash/4.1 level reasoning is required
Use BAA available (Vertex AI HIPAA eligible) (Vertex AI supports HIPAA)
Skip You need Need Azure features (see azure-native)
Skip Multi-cloud is a strategic requirement (see 01d)
